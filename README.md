# 📁 Personal Portfolio Website

HTML, CSS, Vanilla JS만으로 만든 싱글 페이지 포트폴리오 웹사이트입니다.

---

## 🛠️ 기술 스택

- **HTML5** — 시맨틱 마크업
- **CSS3** — CSS 변수, Flexbox, Grid, backdrop-filter
- **Vanilla JavaScript** — 스크롤 이벤트, Clipboard API

외부 라이브러리나 프레임워크 없이 순수 웹 기술로 구현했습니다.

---

## ✨ 구현 기능

- 다크 모드 기반 디자인 (Slate + Indigo 컬러 팔레트)
- 스크롤 진행률 바
- `backdrop-blur` 스티키 헤더
- 타임라인 형식의 프로젝트 섹션
- 이메일 클립보드 복사
- `@media print` 인쇄 대응

---

```bash
git clone https://github.com/Chae102/portfolio.git

open index.html
```
